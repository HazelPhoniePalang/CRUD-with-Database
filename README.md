# PHP CRUD Application with Database

## Overview
This project is a simple **PHP CRUD (Create, Read, Update, Delete)** application connected to a MySQL database. It demonstrates basic server-side data management using PHP and SQL.

The system allows users to:
- Create new records  
- Read/view existing records  
- Update existing records  
- Delete records  

---

## Technologies Used
- PHP (7.4 or higher recommended)
- MySQL / MariaDB
- HTML / CSS (for user interface)
- Apache/Nginx (XAMPP, WAMP, or LAMP)
