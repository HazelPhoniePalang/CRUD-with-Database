# PHP CRUD Application with Database

## Overview
This project is a simple **PHP CRUD (Create, Read, Update, Delete)** application connected to a MySQL database. It demonstrates basic server-side data management using PHP and SQL.

The system allows users to:
- Create new records  
- Read/view existing records  
- Update existing records  
- Delete records  

---

## Technologies Used
- PHP (7.4 or higher recommended)
- MySQL
- HTML / CSS / Bootstrap
- Apache (XAMPP)

---

## Purpose

This project is intended for:

- Learning PHP database connectivity  
- Understanding CRUD operations  
- Practicing backend web development fundamentals
